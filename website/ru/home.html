Привет,
Меня зовут OxyBot, я многофункциональный бот написанный FreeAppSW на Discord.js.

Мой основные функции включают:

  • Команды поиска информации (поиск картинок, поиск в MediaWiki API)
  • Команды получения информации из Discord API
  • Бесшумная система рейтингов пользоваетелей и публичные страницы для серверов
  • Система пользовательских команд с большим диапазоном возможностей (добавить роль, убрать роль, отправить сообщение, отправить в ЛС)
  • Элементы музыкальных команд
  • Скоро Система Gateways для соединения нескольких текстовых каналов
  • Скоро Защита от спам-ботов
