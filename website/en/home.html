I am OxyBot, a multipurpose Discord.js bot made by FreeAppSW.

My main features include:

  • Information API commands (Image search, MediaWiki API search)
  • Discord information commands (guild, user, channel info)
  • Silent leaderboards and guild public pages
  • Custom actions system with multiple features (add role, remove role, send message, send a DM)
  • Basic music features
  • Coming Soon Gateways to link multiple channels together
  • Coming Soon Anti-spam-bot protection for your guilds
