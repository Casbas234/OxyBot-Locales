Bonjour,
je suis OxyBot, un robot polyvalent créé avec Discord.js bot dirigé par FreeAppSW.

Mes fonctionnalitées principales sont:

  • Commandes d'information API (Recherche d'images, recherche API MediaWiki)
  • Commandes d'informations discord (serveur, utilisateur, cannaux, info)
  • leaderboards et pages publiques du serveur Silencieux
  • Actions personnalisés du system avec plusieurs fonctionnalitées (ajouter des rôles, enlever des rôles, envoyer des messages, envoyer un message privé)
  • Fonctionnalités de musique basiques
  • Bientôt Pont pour lier plusieurs canneaux entre eux
  • Bientôt Protection Anti-spam-bot pour votre serveur
